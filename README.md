# 📃 프로젝트 소개

Spring Security와 소셜로그인 api, 주소검색 api를 활용한 한영 주소 검색 서비스입니다.

회원가입을 한 후 주소를 검색 할 수 있으며, 선택한 주소에 대하여 영문 주소까지 함께 볼 수 있는 서비스 입니다.
